# primomedico
«PRIMO MEDICO s'est fixé lui-même pour objectif de donner un visage à la médecine de haute qualité dans tout l'espace germanophone et de la rendre disponible gratuitement pour les patients résidents sur le territoire ou à l'étranger.»
